# Análisis de Coches Usados en EE.UU.

## Descripción del Proyecto
Esta aplicación web permite explorar y analizar un conjunto de datos de anuncios de venta de coches usados en Estados Unidos de manera interactiva.

## Funcionalidad
La aplicación proporciona las siguientes características:
- Visualización de histogramas para analizar la distribución del odómetro de los vehículos
- Gráficos de dispersión para explorar la relación entre el odómetro y el precio
- Botones interactivos para generar visualizaciones bajo demanda
- Interfaz intuitiva desarrollada con Streamlit

## Librerías utilizadas
- `pandas` - para el manejo y análisis de datos
- `plotly.express` - para crear visualizaciones interactivas
- `streamlit` - para desarrollar la interfaz web

## Cómo ejecutar la aplicación
1. Clona este repositorio
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta la aplicación: `streamlit run app.py`
4. Abre tu navegador en `http://localhost:8501`
